# 中世纪二次元竞拍世界观与美术方案

> **文档版本**：v2.0（10 名代理人 · 与 `config/characters.json` / `character_skills.gd` 实现一致）

## 1. 核心世界观

游戏发生在「圣契王庭」崩塌后的边境商都「琥珀港」。王国旧贵族、佣兵团、魔法学院、地下公会与黑甲骑士团都在争夺遗迹仓库中的古代宝物。玩家不是传统冒险者，而是被邀请进入「王冠竞拍会」的**代理人**：每一轮竞拍都像一次没有硝烟的决斗，金币、情报、心理压迫与时机判断决定谁能带走真正有价值的藏品。

世界气质：华丽但危险 — 中世纪宫廷、魔法拍卖行、地下黑市、冒险者公会混合。视觉上有二次元角色的明亮吸引力，同时保留金属、羊皮纸、宝石、暗纹、蜡封和魔法光效。

## 2. 统一视觉关键词

- 题材：中世纪幻想、王庭拍卖、遗迹宝物、冒险者公会
- 角色：二次元半身头像，精致面部，强职业与派系识别
- 氛围：高价值竞拍、暗藏博弈、华丽但克制
- 画风：高质量日系手游角色立绘，干净线稿，柔和赛璐璐上色，局部厚涂质感
- UI：暗色石板与深木底，金属镶边，宝石高光，羊皮纸信息层，魔法蓝绿色提示光
- 主色：深靛蓝 `#182033`、王冠金 `#D8A84E`、羊皮纸 `#E7D4A2`、宝石青 `#40C2B8`、酒红 `#8B2F45`
- 禁忌：现代街机风、科幻霓虹、Q版低幼、过度厚重写实、性感擦边、杂乱大面积花纹

### 3.1 战士：艾莉娅·狮心

- **id**：`aria_lionheart`
- **身份**：前王庭近卫，现任竞拍会护卫代理人。相信宝物应落入能保护王国的人手中，在竞拍桌上像持盾决斗一样稳健。
- **气质**：稳健、持盾决斗、秩序优先

**头像要点**：蜜金发，蓝眼，白银轻甲，红色短披风，狮纹肩甲，坚定微笑；背景王庭旗帜与暖金光（虚化）。

**立绘路径**：`res://assets/ui/characters/portrait_warrior_aria_lionheart.png`

**局内技能：王庭宣价**（`court_announce`）

- **效果（已实现）**：每轮随机翻开 **2 件**藏品的**全部信息**（名称、品质、占格、类型、估值），写入情报列表并在战利品格中揭示。
- **玩法定位**：稳定的信息增量，适合需要逐轮看清关键件的玩家。

**包装参考**：档案标题《王庭近卫 · 代理人委任状》；蜡封王冠金 + 狮纹；标语「货单可假，盾不会」。

## 5. UI 风格方向

整体称为「王冠竞拍会」。界面应像玩家坐在中世纪幻想拍卖大厅内操作，而不是普通表格。

### 5.1 主要界面底图

- **拍卖大厅**：深木长桌、拱形窗、吊灯、黄金拍卖槌、宝物魔法投影；中部留信息区，边缘石柱与帷幕。
- **仓库格子**：石板/羊皮纸格网，暗金描边，稀有度宝石光。
- **结算界面**：王冠契约、金币堆、蜡封账单、胜利角色剪影。

### 5.2 控件与头像框

- **主按钮**：深蓝石板底，金属边框，按下宝石青发光
- **加价按钮**：金色金属，拍卖槌图标
- **放弃/关闭**：酒红底，暗金边
- **输入框**：羊皮纸底，深棕字，细金边
- **头像框**：职业纹章 + 金属环；领先者加王冠小标；十位代理人可按派系蜡封色区分描边
- **道具格**：石板槽 + 稀有度宝石角标
- **情报列表**：羊皮纸标题条 + 半透明深蓝正文区（战术/角色技能条目共用）

### 5.3 代理人档案卡（选角/解锁用）

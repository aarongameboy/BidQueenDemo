# 中世纪二次元竞拍世界观与美术方案

## 1. 核心世界观

游戏发生在「圣契王庭」崩塌后的边境商都「琥珀港」。王国旧贵族、佣兵团、魔法学院、地下公会与黑甲骑士团都在争夺遗迹仓库中的古代宝物。玩家不是传统冒险者，而是被邀请进入「王冠竞拍会」的代理人：每一轮竞拍都像一次没有硝烟的决斗，金币、情报、心理压迫与时机判断决定谁能带走真正有价值的藏品。

世界气质：华丽但危险，像中世纪宫廷、魔法拍卖行、地下黑市和冒险者公会混合在一起。整体视觉应有二次元美少女角色的明亮吸引力，同时保留竞拍游戏需要的金属、羊皮纸、宝石、暗纹、蜡封和魔法光效。

## 2. 统一视觉关键词

- 题材：中世纪幻想、王庭拍卖、遗迹宝物、冒险者公会
- 角色：二次元美少女，半身头像，精致面部，强职业识别
- 氛围：高价值竞拍、暗藏博弈、华丽但克制
- 画风：高质量日系手游角色立绘，干净线稿，柔和赛璐璐上色，局部厚涂质感
- UI：暗色石板与深木底，金属镶边，宝石高光，羊皮纸信息层，魔法蓝绿色提示光
- 主色：深靛蓝 `#182033`、王冠金 `#D8A84E`、羊皮纸 `#E7D4A2`、宝石青 `#40C2B8`、酒红 `#8B2F45`
- 禁忌：现代街机风、科幻霓虹、Q版低幼、过度厚重写实、性感擦边、杂乱大面积花纹

## 3. 五位角色设定与局内技能

### 3.1 战士：艾莉娅·狮心

身份：前王庭近卫，现任竞拍会护卫代理人。她相信宝物应落入能保护王国的人手中，因此在竞拍桌上像持盾决斗一样稳健。

头像要点：金发或蜜金发，蓝眼，半身近景，白银轻甲，红色短披风，狮纹肩甲，坚定微笑，背景有王庭旗帜和暖金光。

局内技能：`王庭宣价`

效果建议：每局 1 次，在任意竞拍轮开始前发动。本轮你的首次有效加价只需满足 `当前最低有效价 - 1 个加价步进`，但最低不能低于起拍价。若本轮最终未成交，技能不返还。

玩法定位：降低关键轮入场门槛，适合保守玩家抢占领先位。

### 3.2 刺客：莉赛特·夜鸦

身份：地下情报公会「夜鸦账簿」的执行人。她不喜欢正面抬价，更擅长在竞拍前知道哪个箱子藏着真正值钱的东西。

头像要点：黑紫长发，金色或紫色眼睛，轻皮甲，兜帽半披，细匕首，鸦羽发饰，表情冷静狡黠，背景是月光下的拍卖廊柱。

局内技能：`夜鸦窥价`

效果建议：每轮竞拍开始前自动翻开 2 个随机未公开道具的类型或品质提示，持续到本轮结束；每局最多触发 3 轮。

玩法定位：情报优势，适合判断是否抬价或放弃。

### 3.3 法师：塞蕾娜·星塔

身份：星塔学院的估价魔导师。她会用星盘和魔法折光读取遗物残留的魔力，让模糊价值变得更可靠。

头像要点：银蓝长发，星空渐变法师帽，水晶法杖，魔法书，明亮青蓝眼，背景有星图、漂浮符文和拍卖会玻璃穹顶。

局内技能：`星盘估值`

效果建议：每局 2 次，在竞拍窗口内发动。立刻获得当前仓库总估值的一个区间提示，例如「偏低 / 合理 / 偏高」或显示相对起拍价的倍率区间。

玩法定位：估价修正，帮助玩家避免情绪化追高。

### 3.4 辅助：米娅·圣铃

身份：圣铃修会的后勤官，负责为冒险者保存契约和赎回资金。她温柔但精明，最懂如何让同伴在下一轮仍有余力。

头像要点：浅粉或浅棕发，绿色眼睛，修女与药师混合服饰，铃铛胸针，治疗法杖，小型账簿，柔和笑容，背景有暖色烛光和羊皮纸契约。

局内技能：`圣铃缓付`

效果建议：每局 1 次，当你成为当前最高价后发动。本轮若成交，你只支付成交价的 85%，剩余 15% 在结算收益中抵扣；若收益不足则照常扣银币。

玩法定位：资金周转，适合中高价值局承压竞拍。

### 3.5 黑骑士：诺克缇丝·黑蔷薇

身份：失落王国的黑甲继承者。她进入竞拍会不是为了财富，而是寻找封印在遗物中的王冠碎片。她的压迫感会让其他竞拍者犹豫。

头像要点：白发或黑银渐变发，红眼，黑色哥特板甲，黑蔷薇纹章，破损黑披风，冷淡表情，背景有暗红月光和黑铁王座剪影。

局内技能：`黑蔷薇威压`

效果建议：每局 1 次，在竞拍窗口开始时发动。本轮所有对手的下一次有效加价额外提高 1 个 `min_raise`；若没有对手加价，成交仍按正常规则进行。

玩法定位：节奏干扰，让自己领先后更容易守价。

## 4. 头像统一规格建议

- 尺寸：`1024x1024` 概念图，正式入库可压缩到 `512x512`
- 构图：胸像近景，头肩占画面 70%，脸部清晰
- 背景：职业符号化环境，低细节虚化，不能抢角色
- 边缘：保留足够安全边距，适配圆形或方形头像裁切
- 表情：每人一眼能读出性格，不做夸张表情包
- 输出：PNG，正式替换前建议统一做圆角蒙版或金属头像框

## 5. UI 风格方向

整体叫做「王冠竞拍会」。界面应像玩家坐在中世纪幻想拍卖大厅内操作，而不是看一张普通表格。

### 5.1 主要界面底图

拍卖大厅：深木长桌、远处拱形窗、吊灯、黄金拍卖槌、展示仓库宝物的魔法投影。中间留出 UI 信息区，边缘保留石柱和帷幕氛围。

仓库格子：石板或羊皮纸格网，宝物格子使用暗金描边，稀有度用宝石光区分。

结算界面：王冠契约、金币堆、蜡封账单、胜利角色剪影。

### 5.2 控件替换方向

- 主按钮：深蓝石板底，金属边框，按下时宝石青发光
- 加价按钮：金色金属按钮，拍卖槌小图标
- 放弃/关闭按钮：酒红底，暗金边
- 输入框：羊皮纸底，深棕字，细金边
- 头像框：职业纹章 + 金属环，领先者显示王冠小标
- 道具格：石板槽 + 稀有度宝石角标
- 提示气泡：半透明深蓝底，羊皮纸标题条
- 进度/倒计时：魔法沙漏或燃烧蜡烛样式

## 6. 推荐首批资源清单

1. 5 张角色头像：战士、刺客、法师、辅助、黑骑士
2. 1 张拍卖大厅主背景
3. 1 张仓库/道具格背景
4. 1 张结算契约背景
5. 1 套 UI 控件风格参考图：按钮、输入框、面板、头像框、道具格、倒计时条

## 7. 审批建议

先审批三件事：

1. 角色气质是否符合你的目标玩家审美。
2. 技能是否符合你想要的竞拍博弈强度。
3. UI 是偏「华丽手游」还是偏「清晰桌游」。当前方案介于两者之间，偏清晰桌游，方便后续真正进游戏。
